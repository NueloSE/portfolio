About

I’m a passionate front-end web developer skilled in crafting responsive, user-friendly web applications using React, TypeScript, JavaScript, and TailwindCSS. With a foundation in HTML, CSS, and Git from my NYSC service at Colab (2024), I trained beginners in web development and explored Web3, began open source contributions through Onlydust to Starknet ecosystem.

Through the ALX Frontend Development Program (2025), I honed my skills, creating a GitHub profile explorer and an ongoing voting platform with Firebase. I thrive in collaborative, fast-paced environments, leveraging problem-solving and mentoring skills to deliver impactful solutions. My goal is to build innovative, optimized, accessible web experiences while growing in Web development. Check out my projects to see my work in action, and let’s connect to create something amazing!
